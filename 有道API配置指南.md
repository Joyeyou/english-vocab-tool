## 📝 申请步骤（10分钟完成）

### 第一步：注册有道智云账号

1. 访问有道智云平台：https://ai.youdao.com/
2. 点击右上角"注册/登录"
3. 使用手机号或邮箱完成注册
4. 登录成功后进入控制台

### 第二步：创建应用

1. 在控制台页面，找到**"自然语言翻译"**服务
2. 点击**"创建应用"**
3. 填写应用信息：
   - **应用名称**：`英语学习工具`（随便填）
   - **应用类型**：选择"文本翻译"
   - **接入方式**：选择"API"
4. 点击**"确定"**创建应用

### 第三步：获取密钥

创建应用后，您会看到：
- **应用ID（AppKey）**：一串字母数字组合
- **应用密钥（AppSecret）**：另一串字母数字组合

**⚠️ 重要：请妥善保管这两个密钥，不要泄露给他人！**

## 🔧 本地配置（用于本地开发）

### 方法：配置环境变量文件

1. 在项目根目录找到 `.env.local.example` 文件
2. 复制一份并重命名为 `.env.local`
   ```bash
   cp .env.local.example .env.local
   ```
3. 用文本编辑器打开 `.env.local`
4. 填入您的密钥：
   ```
   YOUDAO_APP_KEY=你的AppKey
   YOUDAO_APP_SECRET=你的AppSecret
   ```
5. 保存文件
6. 重启开发服务器：
   ```bash
   # 停止当前服务器（Ctrl + C）
   npm run dev
   ```

### ✅ 验证配置

重启服务器后，输入任意单词（如 `abandon`），如果显示正确的释义，说明配置成功！

## 🌐 Vercel部署配置

### 如果已部署到Vercel

1. 登录 [Vercel控制台](https://vercel.com/)
2. 进入您的项目
3. 点击 **Settings（设置）**
4. 找到 **Environment Variables（环境变量）**
5. 添加两个变量：

   | Name | Value |
   |------|-------|
   | `YOUDAO_APP_KEY` | 你的AppKey |
   | `YOUDAO_APP_SECRET` | 你的AppSecret |

6. 点击 **Save（保存）**
7. 回到 **Deployments** 页面
8. 点击右上角的 **Redeploy（重新部署）**
9. 等待部署完成（约1-2分钟）

### ✅ 验证部署

访问您的Vercel网址，输入任意单词测试。

## ❓ 常见问题

### Q1: 我没有信用卡，能申请吗？
**A:** 可以！有道API不需要绑定信用卡，注册即可免费使用。

### Q2: 如何查看我的使用量？
**A:** 登录有道智云控制台，在"数据统计"中查看API调用次数。

### Q3: 超过免费额度怎么办？
**A:** 超过后API会返回错误，但不会扣费。等待下一个周期（小时/天）恢复即可。对于学习来说，免费额度完全够用。

### Q4: 配置后还是显示模拟数据？
**A:** 请检查：
1. 密钥是否复制完整（没有多余空格）
2. 环境变量名是否正确（区分大小写）
3. 是否重启了开发服务器
4. 浏览器控制台（F12）是否有API错误提示

### Q5: 能在手机上配置吗？
**A:** 手机访问部署好的网站即可使用，配置需要在电脑上完成。
