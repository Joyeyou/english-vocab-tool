# v0.3.0 重大更新说明 🎉

## 🚀 核心升级

### 集成Free Dictionary API
我们现在使用**完全免费**的Free Dictionary API，提供专业级词典功能！

## ✨ 新功能对比

### 升级前 (v0.2.0)
❌ 只显示简单翻译
```
释义：放弃

例句：通用学习例句
```

### 升级后 (v0.3.0)
✅ 完整的词典信息
```
音标：/əˈbæn.dən/

释义：
v. 放弃
英文：To give up or relinquish control of, to surrender...

常见短语：
同义词（意思相近的词）

例句：
"Many baby girls have been abandoned on the streets of Beijing."
示例：这里"abandon"表示"放弃"的意思。
```

## 📊 详细改进

### 1. 音标显示 ✅
- **之前**：无音标显示
- **现在**：完整的国际音标，如 `/əˈbæn.dən/`

### 2. 词性标注 ✅
- **之前**：无词性
- **现在**：清晰标注 v. (动词), n. (名词), adj. (形容词), adv. (副词)

### 3. 释义质量 ✅
- **之前**：只有简单的中文翻译
- **现在**：
  - 中文翻译（来自有道API）
  - 详细英文释义（专业词典定义）
  - 双语对照学习

### 4. 真实例句 ✅
- **之前**：通用模板例句
- **现在**：来自真实语料库的地道例句

### 5. 同义词 ✅
- **之前**：无
- **现在**：显示相关同义词，帮助扩展词汇量

## 🔧 技术优化

### 智能降级策略
```
第一优先级：Free Dictionary API (完整词典)
    ↓ 失败
第二优先级：有道翻译API (基础翻译)
```
