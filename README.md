# 英语单词学习工具 📚

专为高二学生准备2027年春考英语而设计的在线单词学习工具。支持单词查询、生词本管理、历史记录、PDF导出和打印功能。

## ✨ 功能特性

- **📝 智能查询**：支持单个或批量查询单词（空格/逗号分隔）
- **💳 单词卡片**：精美的卡片式设计展示单词详细信息
  - 词性和中英文释义
  - 常见短语搭配
  - 例句示范
- **⭐ 生词本**：一键收藏重要单词，随时复习
- **📖 历史记录**：自动保存查询历史，方便回顾
- **📄 PDF导出**：将单词卡片导出为PDF文件
- **🖨️ 打印功能**：直接打印单词卡片
- **💾 本地存储**：使用浏览器本地存储，无需账号

## 🚀 快速开始

### 本地开发

1. **克隆或下载项目**
```bash
cd english-vocab-tool
```

2. **安装依赖**
```bash
npm install --cache /tmp/npm-cache
```

3. **配置环境变量（可选）**
```bash
# 复制环境变量模板
cp .env.local.example .env.local

# 编辑 .env.local 文件，填入有道API密钥
# 如果不配置，将使用内置的模拟数据
```

4. **启动开发服务器**
```bash
npm run dev
```

5. **访问应用**
打开浏览器访问：`http://localhost:3000`

### 获取有道API密钥（可选）

如果想使用真实的词典数据，需要申请有道翻译API：

1. 访问 [有道智云AI开放平台](https://ai.youdao.com/)
2. 注册账号并登录
3. 进入控制台创建应用
4. 获取 AppKey 和 AppSecret
5. 将密钥配置到 `.env.local` 文件中

> **注意**：有道API提供免费额度，一般个人使用完全足够。如果不配置API密钥，应用会使用内置的模拟数据（包含hello、study、learn等常见单词）。

## 🌐 部署到Vercel

### 方法一：通过Vercel CLI

1. **安装Vercel CLI**
```bash
npm install -g vercel
```

2. **登录Vercel**
```bash
vercel login
```

3. **部署项目**
```bash
cd english-vocab-tool
vercel
```

4. **配置环境变量**（如果使用有道API）
在Vercel项目设置中添加环境变量：
- `YOUDAO_APP_KEY`
- `YOUDAO_APP_SECRET`

### 方法二：通过Vercel网站

1. 访问 [Vercel](https://vercel.com/)
2. 登录并点击 "Add New Project"
3. 导入项目（如果已推送到GitHub，可以直接导入仓库）
4. 在项目设置中配置环境变量
5. 点击 "Deploy" 开始部署

## 📱 使用说明

### 查询单词

1. 在搜索框中输入单词（支持多个单词，用空格或逗号分隔）
2. 点击"查询"按钮或按回车键
3. 等待查询结果显示

示例：
- 单个单词：`hello`
- 多个单词：`hello study learn` 或 `hello, study, learn`

### 管理生词本

- 点击单词卡片上的星标图标可以添加/移除生词
- 点击顶部"生词本"标签查看所有收藏的单词

### 查看历史记录

- 点击顶部"历史记录"标签查看所有查询过的单词
- 历史记录自动保存，最多保存100个

### 导出和打印

- **导出PDF**：点击"导出PDF"按钮，将当前显示的所有单词卡片保存为PDF文件
- **打印**：点击"打印"按钮，直接打印当前页面的单词卡片

### 清空数据

- 在不同的标签页点击"清空"按钮：
  - 查询结果：清空当前显示的结果
  - 历史记录：清空所有历史记录
  - 生词本：清空所有收藏的单词

## 🛠️ 技术栈

- **框架**：Next.js 16 (App Router)
- **语言**：TypeScript
- **样式**：Tailwind CSS 4
- **图标**：Lucide React
- **PDF生成**：jsPDF + html2canvas
- **API**：有道翻译API

## 📝 开发笔记

### 变量命名规范
项目遵循小驼峰命名法（camelCase）：
- `wordData`、`inputValue`、`activeTab` 等

### 数据持久化
使用浏览器的 `localStorage` API 存储：
- 历史记录（最多100条）
- 生词本（无限制）

数据格式参见 `types/word.ts`

### API集成
- 支持有道翻译API（需配置密钥）
- 内置模拟数据作为后备方案
- 错误处理和友好提示
