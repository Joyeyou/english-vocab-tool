# 英语单词学习工具 - 使用说明 📖

欢迎使用英语单词学习工具！这是一款专为准备2027春考的高二学生设计的在线单词学习助手。

## 🎯 核心功能

### 1️⃣ 单词查询

**如何使用：**
1. 在顶部搜索框输入单词
2. 支持多种输入方式：
   - 单个单词：`hello`
   - 多个单词（空格分隔）：`hello study learn`
   - 多个单词（逗号分隔）：`hello, study, learn`
3. 按回车键或点击"查询"按钮
4. 等待1-2秒，系统会显示所有单词的详细信息

**单词卡片包含：**
- 📝 单词和音标
- 🔤 词性（n. v. adj. adv. 等）
- 🇨🇳 中文释义
- 🇬🇧 英文释义（如果有）
- 💬 常见短语搭配
- 📚 例句和翻译

### 2️⃣ 生词本

**添加生词：**
- 点击单词卡片右上角的⭐星标图标
- 星标变黄色表示已添加到生词本

**查看生词本：**
- 点击顶部"生词本"标签
- 查看所有收藏的单词
- 再次点击星标可以取消收藏

**使用场景：**
- 遇到难记的单词立即收藏
- 考前集中复习重点词汇
- 制作个人专属单词表

### 3️⃣ 历史记录

**自动保存：**
- 每次查询的单词自动保存
- 最多保存100个历史记录
- 按时间倒序排列（最新的在前）

**查看历史：**
- 点击顶部"历史记录"标签
- 快速回顾之前查过的单词
- 点击星标可以添加到生词本

**清空历史：**
- 点击"清空"按钮
- 确认后删除所有历史记录

### 4️⃣ PDF导出

**如何导出：**
1. 查询或打开历史/生词本
2. 点击绿色"导出PDF"按钮
3. 等待几秒钟处理
4. PDF自动下载到浏览器默认位置

**PDF内容：**
- 包含当前页面显示的所有单词卡片
- 保持卡片的完整格式和样式
- 文件名：`单词卡片-日期.pdf`

**使用场景：**
- 打印学习资料
- 制作复习材料
- 分享给同学
- 离线学习

### 5️⃣ 打印功能

**如何打印：**
1. 查询或打开历史/生词本
2. 点击紫色"打印"按钮
3. 浏览器打印对话框会自动弹出
4. 选择打印机和打印设置
5. 点击"打印"完成

**打印优化：**
- 自动隐藏不需要的按钮和输入框
- 优化纸张布局
- 保留完整的单词信息

## 🎓 学习建议

### 高效记忆
1. **理解词性**：不同词性有不同用法
2. **记忆短语**：短语比单词更实用
3. **例句记忆**：在句子中理解单词
4. **定期复习**：遗忘曲线规律复习

### 春考备考策略
1. **高频词汇优先**：先掌握考试常见词
2. **系统学习**：按主题分类学习（如环保、科技、健康等）
3. **真题词汇**：查询历年真题中的生词
4. **每日积累**：每天至少学习20个新单词

## 📞 常见问题

**Q: 为什么有些单词查不到？**
A: 如果使用模拟数据，只有少量示例单词。建议配置有道API获取完整词典数据。

**Q: 数据会丢失吗？**
A: 只要不清除浏览器数据，历史和生词本会永久保存。建议定期导出PDF备份。

**Q: 能在手机上使用吗？**
A: 可以！所有功能都支持手机浏览器，建议使用横屏模式。

**Q: 导出的PDF太大怎么办？**
A: 单词太多时PDF会较大。建议分批导出，比如每次导出20-30个单词。

**Q: 能离线使用吗？**
A: 首次访问后，页面会缓存，但查询新单词需要联网。可以先查询并导出PDF离线学习。
